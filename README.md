mira
====

[Mira](http://www.copysense.co.uk/mira.php) fractal generator written using;

* [Browserify](browserify.org)
* [Pixi.js](http://www.pixijs.com/) 
* [dat.gui](https://github.com/dataarts/dat.gui)

Usage
=====

* npm install 
* gulp dev
* http://localhost:9966
